## Steps to Increase Quota in Azure:
1. Locate Quotas: Sign in to the Azure Portal and search for "Quotas".
1. Select Resource: On the Overview page, select the provider (e.g., Compute, Azure Machine Learning).
1. Request Increase: Select the specific quota name, ensure "Adjustable" is set to "Yes", and click New Quota Request.
1. Set New Limit: Enter a higher numerical value for the new quota limit and select Submit.
